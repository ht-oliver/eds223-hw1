# EDS 223 - Homework 1
Map making practice for EDS 223 - Environmental Injustice in Wayne County, Michigan


## Purpose
The purpose of this repository is to house that data and scripts that produce metrics on environmental injustice in Wayne County, Michigan. Specifically, the report produced in this repository seeks to test the hypothesis that lower-income communities are more likely to be located near hazardous waste disposal sites than their high-income counterparts.

## Contents


#### data/: 
contains raw data used in analysis

######   - ejscreen: raw data for census tracts
   
######   - GL250515_lam: shapefile for Great Lakes
   
######   - tl_2017_26_place: Shapefile for Michigan cities
    
#### ej_screen_files/:
contains files generated by ej_screen.qmd

######   - figure-html: contains figures generated in analysis

######   - libs: contains libraries
    
#### README.qmd:
This README document

#### ej_screen.qmd:
Quarto document which contains all code used in analysis, along with descriptions

#### eds223-hw1.Rproj:
R Project for this analysis
